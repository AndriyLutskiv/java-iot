# Перелік використаних скорочень

| Скорочення | Повна назва | Пояснення |
|------------|-------------|-----------|
| API | Application Programming Interface | Інтерфейс програмування застосунків; набір класів, методів та протоколів для взаємодії між програмними компонентами |
| CLI | Command Line Interface | Інтерфейс командного рядка; спосіб взаємодії з програмою через текстові команди |
| CSV | Comma-Separated Values | Формат зберігання табличних даних у текстовому вигляді з роздільником-комою |
| DB | Database | База даних; організована структура для зберігання, пошуку та обробки даних |
| FSM | Finite State Machine | Скінченний автомат; обчислювальна модель з кінцевою множиною станів та переходів між ними |
| GUI | Graphical User Interface | Графічний інтерфейс користувача; спосіб взаємодії з програмою через графічні елементи |
| HTML | HyperText Markup Language | Мова розмітки гіпертексту; стандартна мова для створення веб-сторінок |
| HTTP | HyperText Transfer Protocol | Протокол передачі гіпертексту; протокол прикладного рівня для обміну даними у мережі |
| I/O | Input/Output | Введення/виведення; механізми читання та запису даних |
| IDE | Integrated Development Environment | Інтегроване середовище розробки; комплексний інструмент для написання, налагодження та тестування коду |
| JDBC | Java Database Connectivity | Стандартний API мови Java для взаємодії з реляційними базами даних |
| JDK | Java Development Kit | Набір засобів розробки Java; включає компілятор, JVM, JRE та стандартну бібліотеку |
| JRE | Java Runtime Environment | Середовище виконання Java; мінімальний набір для запуску Java-застосунків |
| JSON | JavaScript Object Notation | Текстовий формат обміну даними, що базується на парах «ключ — значення» |
| JUnit | Java Unit Testing Framework | Фреймворк для написання та виконання модульних тестів у Java |
| JVM | Java Virtual Machine | Віртуальна машина Java; програмне середовище для виконання байт-коду |
| LTS | Long-Term Support | Версія програмного забезпечення з тривалою підтримкою безпеки та виправлень |
| OOP | Object-Oriented Programming | Об'єктно-орієнтоване програмування; парадигма, що базується на поняттях класів та об'єктів |
| SDK | Software Development Kit | Набір засобів розробки програмного забезпечення для конкретної платформи або технології |
| SQL | Structured Query Language | Мова структурованих запитів; стандартна мова для роботи з реляційними базами даних |
| TCP | Transmission Control Protocol | Протокол управління передачею; надійний протокол транспортного рівня з гарантованою доставкою |
| TCP/IP | Transmission Control Protocol / Internet Protocol | Стек мережевих протоколів, що є основою сучасного Інтернету |
| UDP | User Datagram Protocol | Протокол датаграм користувача; ненадійний, але швидкий протокол транспортного рівня |
| URL | Uniform Resource Locator | Уніфікований вказівник ресурсу; стандартний спосіб адресації ресурсів у мережі |
| UTF | Unicode Transformation Format | Формат перетворення Unicode; стандарт кодування символів (UTF-8, UTF-16) |
