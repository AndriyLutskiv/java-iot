# Лабораторна робота №2
## Операції з примітивними типами даних та операторами. Операції з масивами

---

### Тема

Операції з примітивними типами даних та операторами; операції з одновимірними та двовимірними масивами мови Java.

---

### Мета

Набути практичних навичок використання примітивних типів даних, арифметичних, логічних та побітових операторів, а також одновимірних і двовимірних масивів мови Java шляхом написання та налагодження навчальних програм.

---

### Теоретичні відомості

#### Примітивні типи даних

Java має вісім примітивних типів, що зберігаються безпосередньо у стеку (на відміну від об'єктів):

| Тип | Розмір | Діапазон | Приклад літерала |
|-----|--------|----------|-----------------|
| `byte` | 8 біт | −128 … 127 | `(byte) 100` |
| `short` | 16 біт | −32 768 … 32 767 | `(short) 1000` |
| `int` | 32 біти | −2³¹ … 2³¹−1 | `42`, `0xFF` |
| `long` | 64 біти | −2⁶³ … 2⁶³−1 | `9_000_000_000L` |
| `float` | 32 біти | IEEE 754 | `3.14f` |
| `double` | 64 біти | IEEE 754 | `3.14159` |
| `char` | 16 біт | 0 … 65 535 (Unicode) | `'А'`, `'A'` |
| `boolean` | 1 біт | `true` / `false` | `true` |

Символ підкреслення в числових літералах (`1_000_000`) допускається починаючи з Java 7 і покращує читабельність.

#### Класи-обгортки (Wrapper classes)

Кожному примітивному типу відповідає клас-обгортка: `Integer`, `Long`, `Double`, `Boolean` тощо. Автоупакування (*autoboxing*) і розпакування (*unboxing*) виконуються компілятором автоматично:

```java
Integer a = 42;          // autoboxing: int → Integer
int b = a;               // unboxing:   Integer → int
int max = Integer.MAX_VALUE;   // 2 147 483 647
String s = Integer.toBinaryString(255); // "11111111"
```

#### Оператори

**Арифметичні:** `+`, `-`, `*`, `/`, `%` (остача від ділення), `++`, `--`  
**Порівняння:** `==`, `!=`, `<`, `>`, `<=`, `>=`  
**Логічні:** `&&` (і), `||` (або), `!` (не), `^` (XOR)  
**Побітові:** `&`, `|`, `^`, `~`, `<<`, `>>`, `>>>` (зсув без знаку)  
**Тернарний:** `умова ? вираз1 : вираз2`

Ділення цілих чисел у Java дає ціле: `7 / 2 == 3`. Для отримання дробового результату необхідно перетворення: `7.0 / 2` або `(double) 7 / 2`.

#### Одновимірні масиви

```java
// Оголошення та ініціалізація
int[] temperatures = new int[10];        // нульові значення за замовчуванням
double[] readings = {36.6, 37.1, 36.8}; // ініціалізатор

// Довжина масиву
int len = readings.length; // поле, не метод!

// Перебір
for (int i = 0; i < readings.length; i++) {
    System.out.printf("readings[%d] = %.1f%n", i, readings[i]);
}

// Розширений for (foreach)
for (double r : readings) {
    System.out.println(r);
}
```

Клас `java.util.Arrays` містить методи сортування (`Arrays.sort`), пошуку (`Arrays.binarySearch`) та порівняння (`Arrays.equals`).

#### Двовимірні масиви

```java
// Матриця 3×4
int[][] matrix = new int[3][4];

// Ініціалізація літералом
int[][] identity = {
    {1, 0, 0},
    {0, 1, 0},
    {0, 0, 1}
};

// Перебір
for (int row = 0; row < matrix.length; row++) {
    for (int col = 0; col < matrix[row].length; col++) {
        System.out.printf("%4d", matrix[row][col]);
    }
    System.out.println();
}
```

Java підтримує «зубчасті» (jagged) масиви — рядки можуть мати різну довжину.

### Хід виконання роботи

1. Оголосити та ініціалізувати змінні кожного з восьми примітивних типів; вивести їх значення разом із іменем типу (`getClass()` для обгорток або `typeof` через `var`).
2. Дослідити межі типів: вивести `Integer.MAX_VALUE`, `Integer.MIN_VALUE`, `Long.MAX_VALUE`, `Double.MAX_VALUE`. Перевірити, що відбувається при переповненні (наприклад, `Integer.MAX_VALUE + 1`).
3. Відкрити у браузері клас `TemperatureReading.java` (посилання вище) та проаналізувати, як у ньому зберігаються показання датчика. Визначити типи всіх полів.
4. Реалізувати обчислення відповідно до свого варіанту індивідуального завдання, використовуючи відповідні арифметичні оператори.
5. Проілюструвати роботу побітових операторів: виконати `&`, `|`, `^`, `<<`, `>>` над двома цілими числами, вивести результати у двійковому (`Integer.toBinaryString`) та шістнадцятковому форматі.
6. Оголосити одновимірний масив відповідно до варіанту, заповнити його значеннями (через цикл або літерал), вивести елементи.
7. Виконати операції над масивом: знайти мінімум, максимум, суму, середнє. Порівняти ручний пошук з методами `Arrays.sort` + `readings[0]`.
8. Реалізувати двовимірний масив відповідно до варіанту (матриця, таблиця тощо).
9. Перевірити роботу тернарного оператора на 3–5 різних умовах.
10. Дослідити неявне та явне приведення типів (`int → double`, `double → int`). Перевірити, чи втрачається точність при звуженні.
11. Оформити звіт: включити вихідний код та знімки виводу програми.

### Індивідуальні завдання

Розробити програму мовою Java (один клас, метод `main`). Обсяг: до 80 рядків коду. Усі дані задати у вигляді масивів або констант; введення з `Scanner` вітається, але не обов'язкове.

**Варіант 1.** Задати масив температур (10 значень типу `double`, від -20 до +40) та обчислити: мінімальну, максимальну, середню температури; кількість днів зі значенням вище середнього. Вивести результати та відсортований масив.

**Варіант 2.** Задати масив цілих чисел (15 елементів) та знайти: суму елементів на парних позиціях; суму на непарних позиціях; кількість від'ємних елементів; добуток ненульових елементів.

**Варіант 3.** Задати матрицю 4×4 цілих чисел. Обчислити: суму головної діагоналі; суму побічної діагоналі; максимальний елемент кожного рядка.

**Варіант 4.** Задати масив показань рівня сигналу (значення типу `int`, від 0 до 100). Знайти кількість значень у кожному діапазоні: 0–25, 26–50, 51–75, 76–100. Вивести гістограму символами `#`.

**Варіант 5.** Задати масив частот нот (7 нот октави, тип `double`). Обчислити різницю частот між сусідніми нотами, вивести ноту з найбільшою та найменшою частотою. За зразок взяти [`Note.java`](https://github.com/AndriyLutskiv/java-iot/blob/main/projects/p03-music-box/src/main/java/ua/crowpi/projects/p03/Note.java).

**Варіант 6.** Задати масив довжин сторін трикутників (масив з N пар значень `double a, b, c`). Для кожного трикутника визначити: чи є він дійсним, вид (рівносторонній, рівнобедрений, різносторонній).

**Варіант 7.** Задати двовимірний масив оцінок (5 студентів × 6 предметів, тип `int`, 1–100). Обчислити середній бал кожного студента та кожного предмета. Визначити студента з найвищим середнім балом.

**Варіант 8.** Задати два масиви цілих чисел однакового розміру. Обчислити: скалярний добуток; поелементну суму та різницю (результат — новий масив); кількість позицій, де перший масив більший за другий.

**Варіант 9.** Задати масив байтових значень (тип `byte`, 20 елементів). Для кожного значення вивести: десяткове, двійкове та шістнадцяткове представлення. Знайти значення з максимальною кількістю одиничних бітів.

**Варіант 10.** Задати матрицю 3×3 дійсних чисел. Обчислити транспоновану матрицю та вивести обидві матриці у форматованому вигляді. Перевірити рівність `A[i][j] == A_T[j][i]`.

**Варіант 11.** Задати масив відліків датчика освітленості (значення типу `int`, 24 елементи — по одному на кожну годину доби). Знайти: пікове значення; середнє за «денний» час (8:00–20:00); кількість «темних» годин (значення < 10).

**Варіант 12.** Задати масив рядків-цін у форматі `"99.99"` (10 елементів, тип `String`). Перетворити в `double` (метод `Double.parseDouble`), знайти суму, мінімум, максимум. Вивести відформатований чек.

**Варіант 13.** Задати масив цілих чисел та виконати сортування методом бульбашки (не використовувати `Arrays.sort`). Підрахувати кількість порівнянь та перестановок. Вивести масив до та після сортування.

**Варіант 14.** Задати квадратну матрицю N×N (N від 3 до 6). Перевірити, чи є вона симетричною (`matrix[i][j] == matrix[j][i]`). Обчислити суму верхнього та нижнього трикутників окремо.

**Варіант 15.** Задати масив цілих чисел (20 елементів). Виконати циклічний зсув масиву вправо на K позицій (K від 1 до 5). Вивести масив до та після зсуву. Не використовувати тимчасовий масив великого розміру.

### Контрольні запитання

1. Перелічіть усі вісім примітивних типів Java. Яке значення займає кожен у пам'яті?
2. Чим примітивний тип відрізняється від об'єктного (reference) типу? Де вони зберігаються в пам'яті JVM?
3. Що таке автоупакування (autoboxing) та розпакування (unboxing)? Яка небезпека пов'язана із розпакуванням `null`?
4. Що відбудеться при виконанні `int x = Integer.MAX_VALUE + 1`? Поясніть явище переповнення.
5. Яка різниця між `float` та `double`? Коли слід використовувати `BigDecimal` замість них?
6. Що повертає вираз `7 / 2` і `7.0 / 2`? Поясніть механізм неявного приведення типів.
7. Що означає оператор `%` для від'ємних чисел? Наведіть приклад: `-7 % 3`.
8. Для чого служить оператор `>>>` (беззнаковий правий зсув) на відміну від `>>`?
9. Що таке «зубчастий» (jagged) масив у Java? Наведіть приклад ситуації, де він є доцільним.
10. Чим `arrays.length` відрізняється від `String.length()`? Чому одне є полем, а інше — методом?
11. Що робить `Arrays.sort` із масивом об'єктів? Який інтерфейс повинні реалізовувати елементи?
12. Як ініціалізуються елементи масиву за замовчуванням залежно від типу?
13. Що таке `ArrayIndexOutOfBoundsException`? Як запобігти цьому винятку?
14. Поясніть різницю між `==` та `equals` при порівнянні масивів.
15. Як за допомогою побітових операцій перевірити, чи встановлено n-й біт цілого числа?
